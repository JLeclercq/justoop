Justoop

What is justoop

A lightweight library for managing classes in javascript

Why Justoop

It let you to play with classes in Javascript in the meanwhile you are waiting for them to be natively and fully supported by the most popular JavaScript engines as defined in ECMAscript version 6.

It's a library for just class management and nothing more: it will not bother you with colorful widgets, new futuristic venusian philosophy or low-calorie diet

And also because:

  1. It has no performance overhead.

  2. It supports single and multiple inheritance.

  3. It religiously respects the infamous javascript prototype chain.

  4. It's for both the client (browser) and the server (such as node.js, nashorn, rhino and similar animals).

  5. You can use it with any other framework (jquery, angularjs, react, etc.).
